Hướng dẫn sử dụng ứng dụng quản lý tín hiệu giao thông (Windows 7 + Python 3.8)

Bước 1: Mở CMD và chuyển đến thư mục vừa giải nén
  cd đường-dẫn-thư-mục

Bước 2: Cài đặt các thư viện cần thiết
  pip install -r requirements.txt

Bước 3: Chạy ứng dụng
  streamlit run app.py

Bước 4: Trình duyệt sẽ tự động mở. Chọn file Excel để hiển thị.

Chúc bạn thành công!
